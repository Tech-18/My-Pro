What I Do

Network Administration

Designing, configuring, and maintaining LAN/WAN networks, routers, switches, and firewalls using Cisco, MikroTik, and Ubiquiti tools.

Penetration Testing

Conducting vulnerability scans and simulated attacks using Kali Linux tools like Nmap, Metasploit, and Burp Suite.

Cybersecurity Hardening

Securing endpoints, web servers, and cloud services against threats through firewall policies, encryption, and system audits.

Packet Analysis

Analyzing live network traffic and PCAP files using Wireshark and tcpdump to detect anomalies and prevent data leakage.

Security Awareness Training

Educating clients and teams on password hygiene, phishing attacks, and safe browsing to build a human firewall.
